# Open Voice Web Demo

This folder is reserved for the future web demonstrator.

It is intentionally separate from the SDK packages so demo code does not get
mixed into the reusable product surfaces.

Current separation:

- `apps/runtime` = backend runtime SDK
- `sdks/web` = reusable web client SDK
- `demos/web` = demo application that will consume the SDK later

We are not implementing the demo yet.
